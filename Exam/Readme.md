Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. > *Результат:*  
>["hello", "2", "world", ":-)"] -> ["2", ":-)"]  
>["1234", "1567", "-2", "computer science"] -> ["-2"]
